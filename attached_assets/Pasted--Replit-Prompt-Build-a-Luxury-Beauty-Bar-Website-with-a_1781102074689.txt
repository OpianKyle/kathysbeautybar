# Replit Prompt: Build a Luxury Beauty Bar Website with an Appointment Booking System (No Online Payments)

I want you to build a **fully responsive, modern, luxury beauty salon website** for **Kat's Beauty Bar** using the attached branding as inspiration. The design should use **soft blush pink, rose gold, white, and black** with elegant animations and a premium feminine aesthetic.

**This is NOT an eCommerce website. There should be NO online payments, NO checkout, and NO deposit system.** The purpose is simply for customers to book appointments.

---

# Tech Stack

Use:

* React + Vite
* Node.js + Express
* PostgreSQL (or Supabase)
* Prisma ORM
* TailwindCSS
* JWT Authentication for Admin
* Nodemailer (or Resend) for emails
* FullCalendar for appointment management

The project should be production-ready and easy to deploy.

# Variable Service Durations

Each service has its own duration.

Examples:

| Service       | Duration |
| ------------- | -------- |
| Cut           | 30 min   |
| Wash Blow     | 45 min   |
| Wash Cut Blow | 60 min   |
| Nanoplastia   | 180 min  |
| Lashes        | 90 min   |
| Nails         | 75 min   |

The system must calculate occupied time automatically.

Example:

If someone books:

10:00

Nanoplastia

(3 hours)

Then the following time slots become unavailable until 13:00.

The booking engine must prevent overlapping appointments.

---

# Live Availability

Customers should only see available appointment times.

Display:

🟢 Available

🔴 Booked

⚪ Closed

Unavailable slots must be disabled.

Availability should update immediately after a booking.
